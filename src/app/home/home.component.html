Bem-vindo

O Open Encoder é uma solução gratuita para todos aqueles que desejam converter os seus vídeos para um formato web.

Com a nossa API de conversão, você pode enviar um link para que todos os seus amigos vejam os seus videos diretamente do navegador do celular sem precisar de baixar nenhum aplicativo!

É só enviar o seu vídeo e pegar o seu link, simples assim !
